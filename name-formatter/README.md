# Name Formatter

Een Python-script dat namen netjes formatteert:

* Hoofdletters voor voor- en achternaam
* Dubbele spaties verwijderen
* Naam netjes combineren

## Gebruik

***bash
python name_formatter.py

---

## Voorbeeld

**Input:**

```
jOhAn   lijFfijt
```

**Output:**

```
Johan Lijffijt
```

## Doel

Praktische tool voor data cleaning en voorbereiding van contactlijsten, CRM-imports of andere tekstverwerking.
