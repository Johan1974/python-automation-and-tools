# Web Scraper

Een eenvoudige Python-tool om data van websites te verzamelen en op te slaan in CSV-bestanden.

## Functionaliteiten

* Data van HTML-tabellen of lijsten scrapen
* Opslaan als CSV
* Flexibel aanpasbaar voor verschillende websites

## Gebruik

+++bash
python web_scraper.py <url> <output.csv>
+++

## Voorbeeld

**Command:**

```
python web_scraper.py https://example.com/products products.csv
```

**Output CSV:**

```
Name,Price,Stock
Widget A,19.99,In stock
Widget B,29.99,Out of stock
```

## Doel

Een snelle manier om gestructureerde data van websites te verzamelen voor analyse, automatisering of rapportages.
